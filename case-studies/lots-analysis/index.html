How It Works

Automated Pipeline

1
Input File Detection
Script scans the input folder for lot Excel files from each 3PL provider. Automatically identifies new files by naming pattern and date.
2
Data Parsing & Normalization
Each provider's file is parsed with openpyxl. Column mappings, date formats, and SKU codes are normalized to a unified schema regardless of source format.
3
Scripting Tracker Integration
Button-level detail is pulled from Scripting Tracker, which captures individual action metadata — quantity, timestamps, operator — and merged into the consolidated report.
4
Reconciliation Logic
Cross-provider lot matching identifies discrepancies: quantity mismatches, missing lot IDs, and duplicate entries. Flagged rows are isolated for review.
5
Consolidated Report Output
Final Excel report generated with per-provider sheets, summary tab, and highlighted discrepancies. Timestamped and saved to the shared output folder.

Core Consolidation Logic

def consolidate_providers(input_dir, output_path):
    """Merge lot reports from all 3PL providers into one report."""
    frames = []

    for filepath in glob(f"{input_dir}/*.xlsx"):
        provider = detect_provider(filepath)   # by filename pattern
        df = parse_lot_file(filepath, provider)  # normalize columns
        df["source"] = provider
        frames.append(df)

    combined = pd.concat(frames, ignore_index=True)

    # Flag discrepancies
    dupes = combined[combined.duplicated("lot_id", keep=False)]
    missing = combined[combined["qty_received"].isna()]
    mismatches = combined[
        combined["qty_expected"] != combined["qty_received"]
    ]

    with pd.ExcelWriter(output_path, engine="openpyxl") as writer:
        combined.to_excel(writer, sheet_name="All Lots", index=False)
        dupes.to_excel(writer, sheet_name="Duplicates", index=False)
        missing.to_excel(writer, sheet_name="Missing Qty", index=False)
        mismatches.to_excel(writer, sheet_name="Mismatches", index=False)

    print(f"✓ Report saved: {output_path}")
    return len(combined), len(mismatches)
